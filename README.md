# Transcribe

A small Python tool that turns long audio recordings into plain-text transcripts using a local copy of OpenAI's Whisper model (via faster-whisper). Everything runs on your machine — no API keys, no uploads.

See full setup and usage instructions at https://port.jp-copeland.com/transcribe

## Quick start

    pip install -r requirements.txt
        python app.py

        Then open http://localhost:5000 in your browser.

        ## Requirements

        - Python 3.9+
        - ffmpeg on your PATH
